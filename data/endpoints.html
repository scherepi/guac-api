Endpoint List:

  • GET / - front-end intro
  • GET /api/hello - returns a simple hello world :)
  • GET /api/djb/CODE - returns a DJB hash for the code provided
  • GET /api/journal - returns the latest journal entry
  • GET /api/journal/[index] - returns the journal entry with the index provided
  • POST /api/journal/ - takes application/json data, writes to the journal whatever's in the "entry" field
